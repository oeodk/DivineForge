Merci d'essayer ce jeu
Ce jeu est encore en développement, ceci la V0.2.0, plus de contenu sera ajouté plus tard (exemple : un vrai titre, de la musique, plus d'ennemis, d'objets et plus).

Au cours d'une partie vous aurez à parcourir des niveaux aléatoires dans le but de vaincre un boss, pour vous aider un objet est présent dans chaque niveau. 

Contrôles :
- Se déplacer : ZQSD
- Attaquer : clic gauche
- Interragir : E
- Ouvrir / Fermer la carte : Tab
- Zoom carte : molette
- Se déplacer sur la carte : ZQSD
- Pause : échap
- Entrer / sortir plein écran : F11 
- Navigation dans les menus à la souris

Si vous trouvez des bugs, avez des idées d'ajouts, des commentaires sur le jeu ou même envisagez de participer au développement du jeu vous pouvez rejoindre ce serveur discord.
